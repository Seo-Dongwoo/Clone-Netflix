# Netflix project

### 주요기능
1. dmdb.com 사이트에서 api를 이용해서 데이터 가져오기
2. 가져온 데이터로 장르별로 영화 목록 만들기
3. originals의 데이털르 이용해서 화면이 업데이트 될 때마다 새로운 메인 컨텐츠 업데이트
4. usestate와 useEffect를 이용해서 상태 관리
